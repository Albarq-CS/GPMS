أقسام وخدمات المنصة الذكية

هندسة برمجية متكاملة تخدم الطالب، المشرف، ولجنة المشاريع في آن واحد.

01

واجهة تسجيل الدخول
Secure Login Gateway

بوابة دخول آمنة وموحدة تتيح لكل مستخدم الوصول إلى صلاحياته ومساحة العمل الخاصة به بناءً على دوره في النظام، مع حماية كاملة لبيانات المستخدمين.

JWT Authentication Role-Based Access Encrypted Sessions
Login Screen
02

بوابة الطالب
Student Portal

مساحة تفاعلية متكاملة تمكّن الطلاب من تشكيل الفرق بالرقم الجامعي، ورفع تسليمات المشروع والكود، وتتبع الملاحظات الأكاديمية والجدول الزمني لحظة بلحظة.

Team Formation File Submissions Live Timeline
03

بوابة لجنة مشاريع التخرج
GP Committee Portal

المركز الإداري للمنصة؛ يتيح للجنة فرز مقترحات المشاريع، واعتماد الفرق، وتشغيل خوارزمية التوزيع الآلي للمجموعات بناءً على العبء التدريسي للمشرفين.

Auto-Assignment Algorithm Proposal Review Load Balancing
Committee Dashboard
04

بوابة المشرف / الدكتور
Supervisor Portal

بيئة عمل رقمية تتيح لأعضاء هيئة التدريس طرح مقترحات المشاريع، ومتابعة أداء المجموعات، ومراجعة التقارير المرفوعة ورصد التقييمات والدرجات بكل سهولة.

Report Reviews Grade Management Project Proposals
Supervisor Dashboard
05

بوابة رئيس القسم
Department Head Portal

واجهة إشرافية عليا توفر نظرة شاملة وتقارير إحصائية دقيقة حول سير مشاريع التخرج في القسم، ونسب الإنجاز, مع صلاحية الاعتماد النهائي للمخرجات.

Analytics Dashboard Final Approval Progress Reports
Head of Department View
06

بوابة مدير النظام
Admin Portal

التحكم الشامل في إعدادات النظام، وإدارة حسابات وصلاحيات المستخدمين، وتهيئة الفصول الدراسية وتحديد المواعيد النهائية (Deadlines) للترم الدراسي.

User Management System Config Deadline Control
Admin Control Panel
07

المساعد الذكي بالذكاء الاصطناعي
AI-Powered Chatbot

نظام دعم ذكي مدمج يعمل على مدار الساعة للإجابة الفورية عن استفسارات الطلاب وأعضاء هيئة التدريس حول لوائح مشاريع التخرج ومواعيد التسليم بدقة وكفاءة.

24/7 Support NLP Processing Smart Answers
AI Assistant Interface
